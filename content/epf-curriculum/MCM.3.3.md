# Module MCM.3.3: Debt Pros/Cons

## Standard: EPF.MCM.3.3

**Objective:** Analyze the advantages and disadvantages of using debt.

---

## Lesson Overview
- Define pros and cons of debt
- Discuss when debt is appropriate
- Analyze real-life debt scenarios

## Teacher Notes
- Use case studies
- Relate to student/parent experiences
- Discuss long-term impacts

## Activities
- Debt scenario analysis
- Debate: Is debt ever good?
- Create a pros/cons chart

## Assessment
- Quiz on debt pros/cons
- Written reflection on debt use
